Nauproject deploy package

Repo GitHub hien dang rong, nen bo source nay duoc tao tu ban Netlify hien tai va da merge them tinh nang giao hang.

Cach dung GitHub:
1. Tai toan bo file trong thu muc nay len repo lequangminh1998br/nauproject.
2. Netlify se deploy tu repo neu da ket noi.

Cach dung Netlify nhanh:
1. Keo tha file nauproject-netlify-deploy.zip vao Netlify Deploys.

Neu dung Google Sheets sync, dan lai google-sheets-apps-script.js vao Apps Script va deploy New version.
